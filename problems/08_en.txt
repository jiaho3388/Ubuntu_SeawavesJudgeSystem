Problem #08: Array Summation

Description:
Write a program that performs the following steps:
1. Prompt the user to enter an integer n, representing the count of numbers to be summed.
2. Prompt the user to enter n integers, which should be stored in an array.
3. Write a function that takes the array as a parameter, calculates the sum of all values within the array, and outputs the result.

Example Input:
3
1 3 5

Example Output:
9
